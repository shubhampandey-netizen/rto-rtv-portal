Import RTO Challans from Excel
Upload a file in the standard format (Date, Invoice number, Product SKU Code, Qty, Unit Price, HSN Code, Customer Name, Shipping Address Line 1/2, City, State, Pincode, GSTIN Number). Rows sharing the same Invoice number are grouped into one challan.
Tax rate defaults to 18% and RTO Reason is left blank on imported challans — edit details afterwards if needed. Item Serial No. is left blank (not present in this format).
